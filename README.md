# cajun-french-api-site

Static landing page for the [Cajun French API](https://cajunfrench.zoasystems.dev) — a community-curated digital lexicon of *le français cadien*.

## Stack

- HTML + [Tailwind CSS](https://tailwindcss.com/) (CDN)
- No build step, no dependencies
- Deployed on [Render](https://render.com) as a static site

## Development

Clone the repo and open `index.html` in a browser. That's it.

```bash
git clone https://github.com/grandtheftdisco/cajun-french-api-site
cd cajun-french-api-site
open index.html
```
